In wrestling, the referee communicates decisions through
hand gestures, which are then manually recorded on a~paper
scoring card -- a~process that slows down the match and is
prone to errors. This thesis explores whether the process
can be simplified using commercially available smartwatches,
without the need for machine learning techniques. The
proposed solution uses an Amazfit T-Rex~3 smartwatch
running Zepp~OS, worn on the referee's left wrist, which
captures motion through its accelerometer and gyroscope.
The data is evaluated in real time by a~companion Android
application using a~threshold-based algorithm calibrated
for eight referee signals defined by the \Gls{uww} rules.
The gestures were designed to follow the referee's natural
mat gesticulation, so that no artificial movements need to
be learned. The reliability of the system was evaluated in
three testing phases, gradually approaching real-world
usage conditions -- from isolated gestures through
a~complete wrestling scenario to testing with external
users without prior experience. The main limitation
identified is the dependence of the threshold-based
algorithm on the individual style of gesture execution,
which could be addressed in the future by introducing
a~calibration phase tailored to each referee.
